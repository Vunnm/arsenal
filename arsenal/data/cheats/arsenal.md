## List global variable
```
>show
```

## Clear all global variable
```
>clear
```

### Set domain global variable
```
>set domain=<domain>
```

## Set DC IP
```
>set dc_ip=<dc_ip>
```

## Set user and password
```
>set user=<user> password=<password>
```

## Set ip global variable
```
>set ip=<ip>
```
