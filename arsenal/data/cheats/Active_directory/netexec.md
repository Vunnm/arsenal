## Extract SMBv1:True list from netexec output

```
cat <nxc_output> | grep "SMBv1:True" | awk '{print "- "$4" ("$2")"}'
```

## Extract signing:False list from netexec output

```
cat <nxc_output> | grep "signing:False" | awk '{print "- "$4" ("$2")"}'
```
